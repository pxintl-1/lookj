# Private Chat - Config Pro Dashboard

一個具備現代黑綠賽博風格的「對話模擬器與字卡控制面板」。

## 🚀 特色功能
* **模組化設計**：HTML、CSS、JavaScript 與數據（JSON）全面分離，易於二次開發。
* **字卡數據管理**：支援手動分組錄入、批量注入，或直接導入標準 JSON 檔案。
* **高自由度模擬**：可自由修改暱稱、頭像、背景，並調整對方的隨機回覆速度及主動發話限制。
* **全螢幕通話模擬**：附帶獨立的語音通話模擬介面與計時器。

## 📂 檔案目錄結構
* `index.html` - 主頁面骨架結構
* `style.css` - 賽博與極簡明暗主題樣式表
* `script.js` - 純前端交互與核心對話模擬邏輯
* `cards.json` - 字卡導入範例數據檔案
* `README.md` - 專案說明文件

## 🛠️ 如何在本地執行？
1. 將此專案 `Clone` 或下載到你的電腦中。
2. 直接雙擊點開 `index.html` 即可在瀏覽器中運行。
3. 點擊頂欄空白處或 ⚙️ 按鈕可開啟「配置控制中心」進行深度設定。
